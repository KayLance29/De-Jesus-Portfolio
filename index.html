Self-Service Ticketing Kiosk for Hotel and Resort Facilities

A web-based self-service kiosk system developed using Bubble.io that enables hotel and resort guests to select services, compute total cost, and generate a QR code for transaction processing. The QR code is scanned by the admin to display the total amount due, and once payment is completed, the admin marks the transaction as paid for accurate and efficient payment tracking.

  • Built using Bubble.io
  • Interactive service selection and booking system
  • QR code generation for transactions
  • Admin verification and payment confirmation
  • Reduces queue time and manual processing
